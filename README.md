# TeamPilot AI

TeamPilot AI is a web-based AI assistant designed to help managers and small teams
organize tasks, track progress, and gain AI-powered insights.

## Features
- Task management for teams
- AI-generated daily summaries
- Team overview dashboard
- Manager-focused insights
- Clean and responsive UI

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Prisma + MySQL
- OpenAI API
- NextAuth
